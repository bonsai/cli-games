:: StoryTitle
知らない駅で降りる

:: Start
電車が止まった。見知らぬプラットフォームに立つ私の目の前で、ドアが閉まる音が響いた。
ここはどこだろう。スマートフォンの地図を開いても、位置情報がうまく表示されない。

[[改札を出る->改札]]
[[そのままホームに留まる->留まる]]

:: 改札
仕方なく改札を出ると、古い商店街が広がっていた。シャッターを下ろした店が多く、人影はまばらだ。
夕暮れの光が路地を長く伸ばしている。

[[商店街を歩く->商店街]]
[[駅に戻る->駅に戻る]]

:: 商店街
「おや、珍しいお客さんじゃないか」
声をかけてきたのは、駄菓子屋の老店主だった。
話を聞けば、この街はかつて賑わったらしいが、今は寂れたとのこと。

[[店主と話す->店主と話す]]
[[店を抜けて先へ進む->先へ]]

:: 店主と話す
「一度降りてみたかったんじゃよ。知らない駅にね」
店主の言葉に、私は笑って頷いた。
店主は懐から古い写真を取り出した。それはかつての賑やかな商店街の姿だった。

[[写真をよく見る->写真]]
[[写真を返す->返す]]

:: 写真
写真の中の子供たちは、今の店主の年齢だっただろうか。
時の流れを感じ、胸が熱くなる。
私もいつか、誰かの思い出の中にだけ存在するようになるのだろう。

[[帰りの電車を待つ->終わり]]

:: 返す
「ありがとう」
店主は写真をしまい、遠い目をした。
その瞳には、かつての賑わいと、静かな諦念が混ざっていた。

[[帰りの電車を待つ->終わり]]

:: 先へ
商店街を抜けると、小さな公園があった。
ブランコがギィギィと音を立て、誰もいない砂場には風が吹いている。
ここにも、かつて子供たちの声が響いていたのだろう。

[[帰りの電車を待つ->終わり]]

:: 留まる
ホームに留まると、次の電車が来るまで十数分ある。
ベンチに座り、あてのない旅を選んだ自分に少し誇らしさを感じた。
窓の外の風景は、まだ見知らぬままだ。

[[次の電車で去る->終わり]]
[[歩いて降りてみる->改札]]

:: 駅に戻る
改札を抜け、再びホームへ。
電車の気配はまだない。
線路の向こうに沈む夕日が、すべてをオレンジ色に染めていた。

[[帰りの電車を待つ->終わり]]

:: 終わり
帰りの電車を待つ間、私はこの街の静かな空気に包まれていた。
知らない駅で降りたことは、他愛のない出来事だ。
けれど、この静寂だけは、確かに私の中に残るだろう。

――了――
